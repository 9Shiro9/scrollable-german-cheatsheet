# German A1.1 — Common Phrases & Expressions (Redewendungen)

> Essential everyday phrases for A1 learners, organized by situation. All phrases include literal translations where helpful.

---

## 1. Greetings & Small Talk (Begrüßung & Smalltalk)

| German | English |
|--------|---------|
| Guten Morgen! | Good morning! |
| Guten Tag! | Good day! / Hello! |
| Guten Abend! | Good evening! |
| Hallo! | Hello! / Hi! |
| Wie geht es Ihnen? | How are you? (formal) |
| Wie geht es dir? / Wie geht's? | How are you? (informal) |
| Mir geht es gut, danke. | I'm doing well, thank you. |
| Mir geht es nicht so gut. | I'm not doing so well. |
| Es geht. | I'm okay. / So-so. |
| Und Ihnen / dir? | And you? |
| Danke, gut. | Fine, thanks. |
| Sehr gut, danke! | Very good, thank you! |

## 2. Introductions (Vorstellung)

| Woher kommst du? | Where are you from? (informal) |

## 4. Politeness & Courtesy (Höflichkeit)

| German | English |
|--------|---------|

| Danke schön! | Thank you very much! |
| Vielen Dank! | Many thanks! |

| Gern geschehen! | My pleasure! |
| Entschuldigung! | Excuse me! / Sorry! |

| Es tut mir leid. | I am sorry. |
| Kein Problem! | No problem! |
| Macht nichts. | Never mind. / It doesn't matter. |
| Bitte? | Pardon? / Could you repeat that? |

## 5. At a Café or Restaurant (Im Café / Restaurant)

| German | English |
|--------|---------|
| Ich möchte bitte ... | I would like ... please. |
| Ich hätte gern ... | I would like ... (more formal) |
| Was möchten Sie trinken? | What would you like to drink? |
| Was empfehlen Sie? | What do you recommend? |
| Die Speisekarte, bitte. | The menu, please. |
| Für mich bitte ... | For me please ... |
| Einmal Kaffee, bitte. | One coffee, please. |
| Zweimal Wasser, bitte. | Two waters, please. |
| Das schmeckt sehr gut! | That tastes very good! |
| Die Rechnung, bitte. | The bill, please. |
| Zusammen oder getrennt? | Together or separately (paying)? |
| Getrennt, bitte. | Separately, please. |
| Zusammen, bitte. | Together, please. |
| Stimmt so. | Keep the change. |

## 6. Shopping (Einkaufen)

| German | English |
|--------|---------|
| Was kostet das? / Wie viel kostet das? | How much does that cost? |
| Das kostet ... Euro. | That costs ... euros. |
| Haben Sie das in Größe ...? | Do you have that in size ...? |
| Haben Sie das in einer anderen Farbe? | Do you have that in another color? |
| Das ist zu groß / klein. | That is too big / small. |
| Das gefällt mir. | I like that. (lit. That pleases me) |
| Das nehme ich. | I'll take that. |
| Wo ist die Kasse? | Where is the checkout? |
| Kann ich mit Karte zahlen? | Can I pay by card? |
| Ich zahle bar. | I pay cash. |
| Haben Sie einen Einkaufswagen? | Do you have a shopping cart? |
| Wo finde ich ...? | Where can I find ...? |

## 7. Asking for Help & Directions (Um Hilfe bitten / Weg fragen)

| German | English |
|--------|---------|
| Können Sie mir helfen? | Can you help me? (formal) |
| Kannst du mir helfen? | Can you help me? (informal) |
| Ich verstehe das nicht. | I don't understand that. |
| Ich spreche kein Deutsch. | I don't speak German. |
| Bitte sprechen Sie langsamer. | Please speak more slowly. (formal) |
| Kannst du das bitte wiederholen? | Can you please repeat that? (informal) |
| Wie sagt man ... auf Deutsch? | How do you say ... in German? |
| Was bedeutet ...? | What does ... mean? |
| Entschuldigung, wo ist ...? | Excuse me, where is ...? |
| Wie komme ich zum/zur ...? | How do I get to the ...? |
| Geradeaus. | Straight ahead. |
| Links. / Rechts. | Left. / Right. |
| Die erste/zweite Straße links. | The first/second street on the left. |
| An der Ampel links/rechts. | At the traffic light, left/right. |
| Ist es weit von hier? | Is it far from here? |
| Das ist in der Nähe. | That is nearby. |
